# @skillsforllms/readme-generator-skill

Auto-write a clean README from project files.

Tier: Free
Launch phase: Tier 2 - Month 2

## Install

```bash
npm install -D @skillsforllms/readme-generator-skill skillsforllms
npx skillsforllms sync @skillsforllms/readme-generator-skill
```

## Usage

Ask your agent to read `.skills/readme-generator-skill/SKILL.md` and follow the conventions while implementing the feature.
