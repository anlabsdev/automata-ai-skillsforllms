# @skillsforllms/project-setup-skill

Scaffold Next.js or Vite with a clean folder structure.

Tier: Free
Launch phase: Tier 1 - Month 1

## Install

```bash
npm install -D @skillsforllms/project-setup-skill skillsforllms
npx skillsforllms sync @skillsforllms/project-setup-skill
```

## Usage

Ask your agent to read `.skills/project-setup-skill/SKILL.md` and follow the conventions while implementing the feature.
