## Install

```bash
npm install -D @skillsforllms/react-component-skill skillsforllms
npx skillsforllms sync @skillsforllms/react-component-skill
```

Ask your agent to read `.skills/react-component-skill/SKILL.md` and follow the conventions while implementing the feature.
